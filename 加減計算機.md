\#\# Problem Description \#\#  
Given two integers A and B, print the following two lines:

1. The result of A \+ B  
2. The result of A \- B

\#\# Input Format \#\#

* A single line containing two integers A and B.  
* Constraint: 0 ≤ A, B ≤ 100

\#\# Output Format \#\#

* The first line should output A \+ B.  
* The second line should output A \- B.

\#\# Sample Input \#\#  
7 3

\#\# Sample Output \#\#  
10  
4
